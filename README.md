# Delivery-System-C-

Реализована система доставки, которая:
- Создает заказы (стандартные и с экспресс-доставкой)
- Управляет меню (есть деление на позиции бара и кухни)
- Обрабатывает изменения статусов заказов
- Уведомляет все подразделения об изменении статуса (то есть динамическая работа)
- Рассчитывает стоимость с доставкой
- Сохраняет меню в формате JSON

# Паттерны в лабораторной:
- Factory Method - для создания разных типов заказов, если мы сразу не знаем тип заказа
- Singleton - для хранения меню в едином экземпляре 
- Observer - для уведомления всех пользователей системы о состоянии каждого заказа
- Strategy - для упраления скидками без изменения логики в основных классах
- Decorator - для добавления системы уведомлений к заказам
